HTML & CSS Basics Summary

HTML Summary

HTML (Hyper Text Markup Language) is used to define our page content,structure and meaning. You don'tuse it for styling purpose. Use CSS for that instead!

  • HTML uses "elements" to describe content.
  • HTML elements typically have an opening tag, content and closing tag
  • You can also have void elements like images.
  • You can also configure elements with attributes.
  • There is a long list of available elements nut you will gain experience over time, no worries!

Learn more about all available HTML elements on the MDN HTML element reference.

CSS Summary

CSS (Cascading Style Sheets) is used for styling our page content.

  • Styles are assigned via property-value pairs.
  • You can assign styles attributes via 'Style' attributes.
  • To avoid code duplication, you typically use global styles instead.
  • Alternatively, you can work with external stylesheet file which you link to.
  • When working with CSS, concepts like "inheritance", "specificity" and the "box model" should be understood.

Learn more about all available CSS properties and values on the MDN CSS property reference.
